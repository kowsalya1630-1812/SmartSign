# SmartSign - AI Powered Sign Language Recognition System

## 📌 Project Overview

SmartSign is an AI-powered web application designed to simulate sign language recognition using webcam integration and interactive dashboard features.

The project helps users understand how AI-based sign language systems work through a modern and responsive interface.

---

## 🚀 Features

- User Login System
- Dashboard UI
- Webcam Integration
- AI Detection Simulation
- Voice Output
- Detection History
- Detection Counter
- Task Management
- User Profile Page
- Settings Page
- Download Detection Report
- Responsive Design

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- MediaDevices API
- Speech Synthesis API

---

## ▶️ How to Run

1. Download the project
2. Open in VS Code
3. Run using Live Server
4. Open browser and explore SmartSign
